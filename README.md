# 实验室网站

左侧导航栏的单页应用（SPA）。点击左侧按钮，右侧内容根据哈希路由切换。

## 使用方法

- 直接双击打开 `index.html`，或使用本地静态服务器打开根目录。
- 侧边栏包含：`概览`、`设备`、`预约`、`联系我们`。

## 结构

- `index.html`：页面骨架与侧边栏。
- `styles.css`：布局与样式。
- `app.js`：哈希路由与页面渲染。

## 自定义

- 在 `app.js` 的 `routes` 中添加新页面。
- 在 `index.html` 侧边 `nav` 中添加对应链接（`href="#/your-route"` 和 `data-route`）。
